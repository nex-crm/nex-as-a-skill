description: What did I miss? Catch-up digest of recent relationship activity
---
Generate a catch-up digest: $ARGUMENTS

**No arguments:** Use mcp__nex__what_did_i_miss with default 7-day window.

**A time period (e.g. "3 days", "this week", "since Monday"):** Use mcp__nex__what_did_i_miss with the appropriate since parameter.

**A person or company name:** Use mcp__nex__search_records to find them, then mcp__nex__get_account_brief or mcp__nex__get_entity_brief for their recent activity.

Format the digest as a clear summary: deals that moved, new interactions, stale deals needing attention, completed tasks, and key changes. Highlight anything that needs immediate action.
